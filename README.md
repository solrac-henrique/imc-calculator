Este projeto foi desenvolvido com o objetivo de **testar meus conhecimentos e colocar em prática** os conceitos  que venho estudando. É o resultado da transição entre a teoria e a aplicação real de lógica de programação e design.

⚙️ Funcionalidades


 Cálculo do IMC a partir de peso e altura informados pelo usuário

Saudação personalizada com o nome do usuário

Classificação automática do resultado com cores:

Abaixo do peso

Peso normal

Sobrepeso

Obesidade grau I, II e III

Validação dos campos de entrada, com mensagens de erro específicas

Seção de FAQ interativa, em formato de acordeão (accordion)

Layout responsivo (funciona em celular e desktop)

HTML5

CSS3 (variáveis CSS, Flexbox, transições)

JavaScript (Vanilla) — manipulação do DOM, validação de formulário, eventoss

Clone este repositório:

Abra o arquivo index.html diretamente no navegador.


🧮 Lógica de cálculo

O IMC é calculado pela fórmula padrão:

IMC = peso (kg) / altura (m)²

O resultado é classificado nas seguintes faixas:

| IMC | Classificação |
|---|---|
| Abaixo de 18,5 | Abaixo do peso |
| 18,5 a 24,9 | Peso normal |
| 25 a 29,9 | Sobrepeso |
| 30 a 34,9 | Obesidade grau I |
| 35 a 39,9 | Obesidade grau II |
| Acima de 40 | Obesidade grau III |

> O IMC é uma referência geral. Para uma avaliação completa, consulte um profissional de saúde.
