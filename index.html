O Índice de Massa Corporal é uma medida que relaciona peso e altura para estimar se uma pessoa está dentro da faixa considerada saudável. Como o cálculo é feito? O IMC é calculado dividindo o peso (em quilos) pela altura (em metros) elevada ao quadrado. O resultado é um diagnóstico médico? Não. O IMC é apenas uma referência geral.
